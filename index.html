HTML - what is it(?)

HTML (from the English HyperText Markup Language — "hypertext markup language") is a standardized markup language for web pages on the World Wide Web. The HTML code is interpreted by browsers; the resulting page is displayed on the screen of a computer monitor or mobile device. The HTML language before version 5 was defined as an application of SGML (the standard generalized markup language according to the ISO 8879 standard). HTML5 specifications are formulated in terms of the DOM (document object model). The strict version of HTML is XHTML, it inherits the XML syntax and is an application of the XML language in the hypertext markup area. HTML pages are usually opened by browsers exchanging information with the server over the HTTP or HTTPS protocol, in plain text or using encryption. The information is taken from here

Javascript - what is it(?)

JavaScript (/ˈdʒɑːvɑːˌskrɪpt/; the streams abbr. JS /ˈdʒeɪ.ɛs./) — multiparadigmatic programming language. Supports object-oriented, imperative, and functional styles. It is an implementation of the ECMAScript standard (ECMA-262 standard[7]). JavaScript is commonly used as an embedded language for programmatic access to application objects. It is most widely used in browsers as a scripting language for interactivity of web pages[8]. Main architectural features: dynamic typing, weak typing, automatic memory management, prototypical programming, functions as first-class objects.The information is taken from here

AngularJS - what is it(?)

AngularJS is an open source JavaScript framework. Designed for the development of single-page applications[5]. Its goal is to extend browser-based applications based on the MVC template, as well as simplify testing and development.The framework works with HTML that contains additional custom attributes that are described by directives, and binds the input or output of a page area to a model that is a regular JavaScript variable. The values of these variables are set manually or extracted from static or dynamic JSON data. The information is taken from here

CSS - what is it(?)

CSS (/siːɛsɛs / English Cascading Style Sheets "cascading style sheets") is a formal language for describing the appearance of a document (web page) written using a markup language (most often HTML or XHTML).
